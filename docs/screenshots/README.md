# Screenshots

Drop PNGs of the running GUI here. The repo's main README links to:

* `gui-factorial.png`  -- the factorial sample loaded, all output tabs populated
* `gui-type-error.png` -- the "string + int" sample, with the Errors tab showing the rejection

To capture:

1. `build.bat`
2. `run.bat`
3. Load the sample from the dropdown.
4. Hit **Compile**.
5. `Win+Shift+S` to snip the window; paste into Paint; save as PNG into this folder.
